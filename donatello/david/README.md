# Donatello David - Form Gallery

## Local viewer file

- `donatello_david_source_small.stl`
  - locally mirrored and decimated from the Commons Scan the World STL
  - 400,000 faces
  - 20,000,084 bytes (~19.1 MB)

## Notes

- The original sculpture is the bronze *David* in the Museo Nazionale del Bargello.
- This route keeps the same native STL viewer and bronze material treatment used elsewhere in Form Gallery.
