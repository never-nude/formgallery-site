# Cecilia Gonzaga - Form Gallery

Sketchfab-backed Form Gallery route for the marble bust identified by the Spis Museum as **Cecilia Gonzaga**, probably by **Donatello**.

Source:
- Sketchfab: https://sketchfab.com/3d-models/ceciliae-gonzagae-sm-7156-9dac3be34f894e94815b7c691e8971f5

The museum description says the bust is made of Carrara marble and is assumed to represent Cecilia Gonzaga; the Donatello attribution is presented as probable rather than certain.
