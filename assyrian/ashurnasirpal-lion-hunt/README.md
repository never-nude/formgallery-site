# Ashurnasirpal II on Lion Hunt - Museum Viewer

Interactive 3D sandbox page for an Assyrian palace relief in the shared ad-arma museum viewer.

- Work title used on page: `Ashurnasirpal II on Lion Hunt (Assyrian palace relief, c. 883-859 BCE)`
- Source institution: SMK Open
- Source record: https://open.smk.dk/artwork/image/KAS1994
- Full STL: https://api.smk.dk/api/v1/download-3d/r207tt80w_smk-kas1994-ashurnasipal-ii-on-a-lion-hunt-decimated.stl

- Viewer note:
  - This route serves a local mirror of SMK's optimized public-domain cast scan for reliability.
  - The underlying original is in the British Museum and comes from the Northwest Palace at Nimrud.
  - The shared viewer uses a relief presentation with no pedestal so the panel reads as architecture rather than a freestanding statue.
