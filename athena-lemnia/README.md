Interactive 3D sandbox page for **Athena Lemnia** in the shared ad-arma museum viewer.

- Backlog item: `318`
- Work title used on page: `Athena Lemnia (copy tradition, c. 450 BCE type)`
- Source institution: SMK Open
- Source record: https://open.smk.dk/artwork/image/KAS40
- Full STL: https://api.smk.dk/api/v1/download-3d/cz30pz12n_20-smk-lemniam-athena-deci.stl

- Viewer note:
  - This route serves a local mirror of SMK's optimized public-domain reconstruction scan for reliability.
  - SMK catalogs the work as Furtwangler's reconstruction of the Athena Lemnia type using the Dresden torso and the Palagi head in Bologna.
  - The viewer uses a bronze material pass to evoke the lost bronze original while keeping the source attribution explicit.
